virtual void Algorithm::enqueue ( )
protectedpure virtual

launch evaluation of the object This function can assumes memory was allocated for all results and that dependencies are enqueued (not necessarily computed).

Note
enqueue should not be blocking execution. Instead, it should start computation in a separate thread. Synchronisation can be achieved with waitEndOfEvaluation() or implementation specific systems.

Implemented in CLMatrixScale, CLMatrixUnloader, InertiaSort< Scalar >, CLInterDistance, CLMatrixLoader, CLMatrixCovariance, CLMatrixNorm, CLMatrixProduct, and EigenPair.
