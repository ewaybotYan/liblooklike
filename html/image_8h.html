ArrayOfImages arrayOfImagesFromFiles ( const std::string  path)

Generates an array of monochomatic images from JPEG files in a directory.

Parameters
pathPath to a directory containing the input files
Returns
A matrix with the pixels of the images, each column corresponds to one image, each column is the concatenation of the lines of pixel from the image.
Note
The directory may contain non JPEG files, they will be ignored.
Warning
Files are read in no specific order
