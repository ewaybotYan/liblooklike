enum ExpressionState

describe the state of an expression (only meaningful if it is a computed expression)

Note
The state of an expression is not public but is used internally in the evaluation process and for integrity checks.
Enumerator
INITIAL 

initial state

ALLOCATED 

memory is allocated to store the value of the expression

ENQUEUED 

the algorithm that compute the value is running

COMPUTED 

the value computation is done, the value can be read
