Matrix load ( std::string  filepath)

Load the values of a text file into a Matrix : all the values must be separated with spaces and these values will be load in the Matrix row by row.

The file must end by the last value of the matrix, or by space(s) or newline(s). Number of rows and columns of the matrix must be indicated at the beginning of the file in a specific way (cf. files.cpp).

Parameters
filepathFilepath with the full name of the text file where the values must be taken to be loaded in a Matrix
Returns
Matrix with the loaded value if there was no problem and throw an error (MissingDataInTheFile or MatrixLoadingIncomplete) if there was
